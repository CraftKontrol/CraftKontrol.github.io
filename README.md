### Getting Started

1. Visit the live site at [https://craftkontrol.github.io](https://craftkontrol.github.io)
2. Enter your Mistral API key (get one at [console.mistral.ai](https://console.mistral.ai/))
3. Start asking questions about CKUI modules

### Topics Covered

- **Project Manager** - Setup, configuration, and usage
- **Library Management** - Installing and managing CKUI libraries
- **Virtual Environments** - Creating and managing Python venvs
- **Logger Integration** - Logging and debugging
- **Git Integration** - Version control setup
- **Dependencies** - Managing project dependencies
